Instructions

Add an event

To add an event click on the day square you wish to add the event to, and select the type of event you would like to add, then click the save button.

Add a custom event

To add a custom event, follow the steps for selecting an event, and from the drop-down menu select “other”. A box will pop up and you can type in the name of the custom event you would like to create.

Add repeating event

To add an event which repeats weekly, in the new event creation pop-up select “repeat weekly”.

Deleting an event

To delete an event, click on the day square you wish to delete an event from, after selecting the “delete event” option, the delete event pop-up will be displayed. The drop-down menu of events follows the same order as the events displayed in the tile. Select from the drop-down menu which event you wish to delete and click the “delete selected event” button.

Deleting all events

To delete an event, click on the day square you wish to delete an event from, after selecting the “delete event” option, the delete event pop-up will be displayed. To delete all of the events for a specific day square click the “Delete All” button. If you wish to delete all of the events on the entire schedule, just refresh the webpage.

Transferring schedule to eClass

Once you have created the schedule and wish to transfer it to class, scroll to the bottom of the page and press the “Save HTML” button followed by the “Copy to Clipboard” button and this will copy the code into your clipboard. Locate the area in eclass you wish to have the schedule, open up the code editor and paste the code into the code editor and press save.
